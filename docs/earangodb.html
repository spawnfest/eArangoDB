    earangodb (eArangoDB v0.1)

    This module provides functions to work with collections, documents and graphs. It is a main API for this library.

    collection_create(Name, CollectionType)
    Creates a new collection with a given name and type.

    Returns an object with an attribute result containing an array of all collection descriptions.
    document_create(CollectionName, Document)
    Creates a new document from the document given in the body, unless there is already a document with the _key given.
    document_delete(CollectionName, DocumentKey)
    Deletes the specified document with a given key.
    document_read(CollectionName, DocumentKey)
    Returns the document identified by document-id. The returned document contains three special attribute _key containing key which uniquely identifies a document in a given collection.
    document_replace(CollectionName, DocumentKey, Document)
    Replaces the specified document with the one in the body, provided there is such a document .
    document_update(CollectionName, DocumentKey, Document)
    Partially updates the document identified by document-id. The body of the request must contain a JSON document with the attributes to patch (the patch document). All attributes from the patch document will be added to the existing document if they do not yet exist, and overwritten in the existing document if they do exist there.

    graph_vertex_update(GraphName, CollectionName, VertexKey, UpdateDoc)

    graphs_create(GraphName, EdgesCollection, FromCollections, ToCollections)
    The creation of a graph requires the name of the graph and a definition of its edges.
    graphs_delete(GraphName)
    Drops an existing graph object by name.
    graphs_get(GraphName)
    Selects information for a given graph. Will return the edge definitions as well as the orphan collections. Or returns a 404 if the graph does not exist.
    graphs_list()
    Lists all graphs stored in this database.
